## :rocket: Technologies 

The project was created using the following technologies:

- NodeJS
- Bootstrap
- React Framework
- CSS
- Firebase Authentication
- Cloud Firestore (NoSQL Database)
- Mongoose
- SVG Map
- MongoDB
- Custom Python Information-Scraping Scripts
- *** All npm packages can be seen in package.json ***
